# Browser Reproduction Debugging

Define URL, browser, viewport, auth state, feature flags, test data, and environment. Reproduce the minimal flow and stop when the bug appears. Capture console errors, network failures, DOM/UI state, screenshots, and relevant redacted request/response shapes. Report exact steps, browser evidence, suspected source path, confidence, and test recommendation.
