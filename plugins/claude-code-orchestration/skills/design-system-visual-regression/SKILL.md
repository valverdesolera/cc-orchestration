# Design System and Visual Regression

Determine expected UI from Figma, Storybook, Chromatic baseline, design-system docs, tokens, variants, and responsive rules. Compare actual browser screenshot/DOM/computed layout by spacing, typography, colors, radii, shadows, variants, breakpoints, alignment, overflow, z-index, empty/loading/error states. Verify component props from docs before conclusions.
