# Observability Correlation

Extract service, environment, timestamp, release, trace ID, request ID, endpoint, error fingerprint, stack trace, span names, and deployment timing. Correlate error spikes with deployments, traffic, dependency latency, DB saturation, queue backlog, config changes, and feature flags. Report evidence, not guesses.
