# Frontend Error Triage

Capture route, component, browser, viewport, console/network errors, screenshot/replay, expected vs actual behavior, environment, release, and regression status. Classify into runtime, rendering, SSR/hydration, routing, state, forms, network/API, auth/session, CSS/layout, accessibility, performance, visual/design, cross-browser/device, or build/test. Reproduce narrowly and report evidence, root cause, confidence, minimal fix plan, and regression test recommendation.
