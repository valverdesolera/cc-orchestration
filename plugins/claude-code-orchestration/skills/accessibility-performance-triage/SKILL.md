# Accessibility and Performance Triage

For accessibility, check labels, roles, focus order, keyboard traps, ARIA state, dialogs, color contrast, reduced motion, and screen-reader-only content. For performance, check long tasks, excessive re-renders, request waterfalls, hydration, bundle size, layout shift, image optimization, cache behavior, memory leaks, and Core Web Vitals. Recommend targeted tests or audits.
