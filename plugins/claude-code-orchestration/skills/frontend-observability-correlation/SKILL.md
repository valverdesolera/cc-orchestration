# Frontend Observability Correlation

Extract error name, route, browser/device, environment, release, timestamp, affected sessions summary, stack trace, breadcrumbs, console logs, network failures, session replay, feature flag, and deployment timing. Correlate by release, browser, route, API failures, user-flow step, locale/timezone, and feature flag. Do not overstate root cause from minified traces without source maps.
